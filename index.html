저는 말이죠

2014

경인여자대학교 스마트미디어과(IT)를 입학했어요.

2015

졸업작품(웹) 전시회에 참여했습니다.

2016

학교를 졸업하고, 산업기사공부를 하고, 포토샵 학원을 다녔어요.

2017

좀더 잘맞고, 좋아하는 일을 찾아보려 동물원, 전망대 등
몇가지 일들에 도전했어요.

2018

서비스업의 한계를 느끼고 사무직을 알아보았고,
IT회사 알바를 시작했어요.

2019

회사에서는 웹 사이트 분석을 하게되었습니다.
퍼블리싱 공부를 결심해서 11개월 다니고 퇴사했어요.

2020

기초 영어공부를 하며 정보를 수집했고,
본격적으로 공부를 시작했어요.

2021

JS공부를 하다가 개발공부로 방향을 수정했어요.
스터디와 동아리를 시작했어요.

이런 것을 할 수 있어요

html/css

시멘틱 태그를 이용해 디자인된 화면을 그대로 구현할 수 있습니다. 반응형 웹을 구현할 수 있습니다.

Javascript

자바스크립트의 기본개념을 이해하고 사용할 수 있으며, 인터렉티브한 화면을 구현할 수 있습니다. 배열, 객체에 쓰이는 함수들을 사용할 수 있습니다.

Typescript

type과 interface의 존재를 알고, 객체지향에 대해 아주 기본적인 부분만 이해하고 있습니다.

React

리액트 사용법을 이해하고 비동기 요청과 상태관리(redux, contextAPI)를 할 수 있고, 디자인 라이브러리를 사용할 수 있습니다.

NodeJs

express를 이용해서 서버를 연결할 수 있고, 프론트와 정보를 주고받는 방식을 이해하고 있습니다. 간단한 REST API를 작성할 수 있습 니다.

Git/Github

기본적인 add, commit, push, fetch, merge, reset 커맨드를 사용할 수 있습니다.

Figma

간단한 디자인을 할 수 있고, 레이어와 컴포넌트에 대해 알고있습니다.

Notion

노션에 공부한 내용들을 정리합니다. 스터디와 동아리 회의를 할 때 이용중입니다. 노션의 갤러리 형태를 좋아합니다.
